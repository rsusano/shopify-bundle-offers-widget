## Editing Tips

1. **Crop tightly**: Remove unnecessary white space
2. **Maintain aspect ratio**: Don't stretch images
3. **Use consistent zoom**: All desktop screenshots at same zoom level
4. **Add annotations** (optional): Arrows, highlights, labels
5. **Compress lightly**: Use TinyPNG.com to reduce file size
6. **Test display**: View on GitHub to ensure clarity
